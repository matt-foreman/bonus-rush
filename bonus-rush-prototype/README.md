# Bonus Rush Prototype

Vite + React + TypeScript prototype for Bonus Rush with React Router and a calm, nature-inspired theme.

## Run locally

1. Install Node.js 18+.
2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Build production bundle:

```bash
npm run build
```

5. Preview production build:

```bash
npm run preview
```

## Routes

- `/` -> Ladder page
- `/puzzle/:puzzleId` -> Puzzle page
- `/results/:puzzleId` -> Results page

## Project structure

- `src/pages` route-level screens
- `src/components` shared UI components
- `src/data` static data and mock payloads
- `src/state` app state management
- `src/types` shared TypeScript types
- `src/utils` utility helpers
- `src/styles` global and thematic CSS
